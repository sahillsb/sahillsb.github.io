DROP YOUR HIDROM FONT FILES HERE
=================================

Expected file names (any of these will work):
  Hidrom.woff2   ← preferred (smallest, fastest)
  Hidrom.woff    ← fallback
  Hidrom.ttf     ← fallback

Once the file is here, rebuild and the font loads automatically.
No code changes needed — @font-face in index.css already points to /fonts/Hidrom.woff2.

If you don't have the font file, the site uses Bebas Neue (Google Fonts) which
has the same heavy condensed display style.
